How to use this site:

First time use

You need input your name, address, and neighborhood when first open this page. Your personal information is stored in your computer, it is not sent or saved in other places. We may collect anonymous reporting data for analysis and will be used to deal with the proper agency. If you run into issue, please try again.

You should also update template

Direct grab data from flightradar24

This is recommended method if you are using desktop PC. Please consider use Chrome, which is well tested. We know there is issue with Firefox, which yet to be fixed. We know this method also works in some Android devices.

We recommend to use https://www.flightradar24.com/37.35,-122.02/12 to check flight details, it is almost real time, with good details. There are other website, such as http://webtrak5.bksv.com/sjc3 which has 20 minutes delay, and can be slow.
